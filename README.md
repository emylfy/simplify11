## ‚ö° Installation

### Quick Start

Launch Simplify11 from PowerShell:

```powershell
iwr "https://dub.sh/simplify11" | iex
```

### Full Installation

Creates a shortcut to Start Menu that launch latest version:

```powershell
iwr "https://dub.sh/s11install" | iex
```
